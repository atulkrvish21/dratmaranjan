# Dr. Atmaranjan Dash Angular Website

Angular 19 upgrade of `dratmaranjandash.com` using standalone components, Tailwind CSS, SCSS, lazy routing, reactive forms, typed content models, and JSON-backed site content.

## Run

```bash
npm install
npm run start
```

Open `http://localhost:4200/`.

## Build

```bash
npm run build
```

## Content

Editable content lives in `src/assets/data/clinic-content.json`. Replace it with API responses through `ClinicContentService` when connecting a backend.
